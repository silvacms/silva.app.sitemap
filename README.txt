=================
silva.app.sitemap
=================

Introduction
============

``silva.app.sitemap`` is an extension let you generate XML sitemaps
0.9 (see http://www.sitemaps.org/protocol.html) for `Silva`_ 3.

A view ``sitemap.xml`` will be available on every folder of your site
in order to access the sitemap.

A local service will permit you to configure which content you want to
see in the sitemap and configure exceptions.


Code repository
===============

Source code can be found in a Git repository at:
https://github.com/silvacms/silva.app.sitemap

.. _Silva: http://silvacms.org
